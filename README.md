A yaml snippets extension for Rasa **3.x**.



## Usage

If Rasa Snippets doesn't work, add following text in settings.json.

```json
{
    "[yaml]": {
        "editor.formatOnSave": true,
        "editor.tabSize": 2,
        "editor.quickSuggestions": {
            "other": true,
            "comments": true,
            "strings": true
        },
        "editor.acceptSuggestionOnEnter": "on"
    }
}
```
